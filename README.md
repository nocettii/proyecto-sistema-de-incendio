# proyecto-sistema-de-incendio
En este proyecto se presenta un plano, que simula la deteccion de un sistema de incendio, que incluye la ubicación de detectores de humo, central de incendios, pulsador manual, repetidores y sirena. Además, se incorpora un programa en Python que simula el funcionamiento de la central de incendios en función de la opción seleccionada.
